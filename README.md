# uCM4

Raspberry Pi Compute Module 4 micro carrier board

# Features

* exactly same size as CM4
* support for both Lite and eMMC variants of CM4
* Gigabit Ethernet
* microSD slot (optional)
* Micro USB for power and USB 2.0 data transfer
* USB-OTG support
